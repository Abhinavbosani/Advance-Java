# Advance Java – Multithreading & Concurrency

This repository contains **Java concurrency and multithreading examples** implemented using core Java APIs.

## Topics Covered
- Thread creation and lifecycle
- Thread pools and Executor framework
- Callable & Future
- Semaphore usage
- Barrier synchronization
- ScheduledExecutorService

## Key Concepts Demonstrated
- Thread safety
- Synchronization mechanisms
- Resource sharing
- Parallel task execution

## Tech Stack
- Java (Core + Concurrency)
- IntelliJ IDEA

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run individual classes from the `src` folder

## Purpose
This project is intended for:
- Learning Java multithreading
- Interview preparation
- Understanding low-level concurrency constructs
